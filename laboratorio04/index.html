Pasos para un algoritmo sencillo secuencial

  1. Se marca el punto de partida
  2. Se recopilan los datos
  3. Se realizan las operaciones y cálculos
  4. Se muestra o almacena el resultado
  5. Se marca el punto final

Pasos para un algoritmo con condiciones y bucles

  1. Se marca el punto de partida
  2. Se recopilan los datos
  3. Se realizan las condicionales para tomar decisiones
  4. Se realizan bucles que recorren los datos
  5. Cada dato se puede mostrar, actualizar o eliminar
  6. Se cierra o sale del bucle
  7. Se marca el punto final
